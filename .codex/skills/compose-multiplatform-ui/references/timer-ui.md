# Timer UI Guidance

## Main Screen

- The remaining time should be the strongest visual element.
- Start, pause/resume, reset, and preset selection should be reachable without hunting.
- Preset changes while running should be disabled or implemented as an explicit reset-and-select product decision.

## Layout

- Use constrained dimensions or stable layout containers for the countdown display.
- Avoid text resizing based on viewport width.
- Ensure long labels fit on small phones.

## Accessibility

- Add content descriptions for icon-only controls.
- Do not rely on color alone for running/paused/finished state.
- Keep touch targets comfortable for repeated kitchen use.
